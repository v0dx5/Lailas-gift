# Website Foundation

A minimal, clean website foundation ready for expansion.

## File Structure

```
├── index.html      # Main HTML file with semantic structure
├── styles.css      # Stylesheet with CSS variables and responsive design
├── script.js       # JavaScript file ready for future features
└── README.md       # This file
```

## Features

- **HTML5** semantic structure
- **CSS Variables** for easy theming
- **Responsive Design** foundation
- **Clean JavaScript** structure ready for features
- **Modern Best Practices** implemented

## Getting Started

1. Open `index.html` in your web browser
2. Start adding your features and content
3. Customize colors and fonts in `styles.css` using CSS variables
4. Add functionality in `script.js`

## Customization

### Colors
Edit the CSS variables in `styles.css`:
- `--primary-color`: Main brand color
- `--secondary-color`: Secondary color
- `--text-color`: Text color
- `--background-color`: Background color

### Adding Features
1. Add HTML structure in `index.html`
2. Style it in `styles.css`
3. Add interactivity in `script.js`
